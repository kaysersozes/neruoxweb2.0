# NeuroX Project

NeuroX is a technology consulting firm dedicated to providing expert services and support for technology platform projects. Our mission is to empower businesses by leveraging cutting-edge technology solutions tailored to their unique needs.

## Project Structure

The project consists of the following files:

- **index.html**: The main entry point of the website, containing the structure and content for the homepage of NeuroX.
- **about.html**: Provides information about the NeuroX consulting firm, including its mission, vision, and team.
- **services.html**: Outlines the various technology consulting services offered by NeuroX, detailing each service and its benefits.
- **contact.html**: Includes a contact form and information for potential clients to reach out to NeuroX.
- **css/styles.css**: Contains the styles for the website, defining the layout, colors, fonts, and overall appearance of the pages.
- **js/main.js**: Includes JavaScript code for interactivity on the website, such as form validation and dynamic content updates.

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the project folder in your preferred code editor.
3. Open `index.html` in a web browser to view the homepage.
4. Navigate through the other pages using the links provided in the navigation menu.
